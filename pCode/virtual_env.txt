Create virtual env :

python -m venv <directory name>

$ python3 -m venv oracledb
$ cd oracledb/bin
$ . activate

=====================================
Select virtualenv for vCode to utilize new packages that has been installed:

ctrl+ship+p -> Python: Select Interpreter -> Enter interpreter path -> select venv python

=====================================
Create virtualenv using python2:

$ virtualenv venv2 --python=python2
